# SpringBoot_Practice
스프링부트 연습용<br>

1. JAVA 및 SpringBoot의 사용.
2. 로그인 구현등 Django를 이용한 프로젝트에서 구현하지 않은 것에 대한 연습.
3. 겸사겸사 깃허브 협업 연습(메세지 작성법)
